# AI Sentiment Analyzer

A web application that classifies customer reviews into Positive, Negative, or Neutral using Machine Learning.

## Tech Stack
- Python (Flask)
- Scikit-learn
- HTML + Tailwind CSS

## Features
- Real-time sentiment prediction
- Confidence score
- Clean Material UI

## How to Run
1. Install dependencies:
   pip install flask scikit-learn pandas

2. Train model:
   python train.py

3. Run backend:
   python app.py

4. Open frontend/index.html
